# Soma Overview

Status: canonical

## Purpose

Soma is the canonical repo for the Soma protocol.

It owns the public package surfaces, normative specs, security model, conceptual rationale, and future protocol evolution work.

## This Repo Owns

- Normative protocol specs
- Security model and explicit limits
- Public package documentation
- Protocol-level explanation and vision
- Future protocol proposals and research directions

## This Repo Does Not Own

- ClawNet product behavior
- ClawNet deployment doctrine
- Pulse product behavior
- Downstream app-specific operational truth

## Related Repos

- `claw-net`: production platform applying Soma in practice
- `pulse`: X.com marketing agent built on ClawNet

## Doc Map

- `docs/reference/`: packages, primitives, and spec index
- `docs/explanation/`: rationale and security framing
- `docs/how-to/`: local development workflow
- `docs/operations/`: release and operational hygiene
- `docs/proposals/`: future protocol and design work
- `docs/archive/`: preserved historical planning artifacts
